# 📸 BlueBooth - Ultimate Edition
> Aesthetic Web Photobooth built with Next.js & Tailwind CSS. No install needed.

Bikin photostrip ala studio langsung dari browser. Didesain dengan UI/UX yang modern, glassmorphism, dan fitur yang relate sama Gen Z trends.

## ✨ Killer Features
- **Live Lens Filters:** Efek iPhone HD, Skena Vintage, dan Classic B&W secara *real-time*.
- **Customizable Layout:** Pilih 3 Grid atau 4 Grid untuk photostrip kamu.
- **Electric Frame Colors:** Palet warna *bold* dan estetik untuk frame foto.
- **Sticker Packs & Watermark:** Tambahkan emoji aesthetic (Coquette 🎀, Y2K Sparkle ✨, Dark Skena 🎱) dan ganti nama watermark sesukamu!
- **Digital Shutter & Flash:** Animasi flash dan suara *shutter* digital ala photobox asli.
- **PWA Ready:** Bisa di-install langsung ke *homescreen* HP-mu!

## 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS (v4)
- **Core Logic:** HTML5 Canvas API & WebRTC (MediaDevices API)
- **Audio:** Web Audio API

## 🚀 Cara Menjalankan Secara Lokal
1. Clone repository ini: `git clone https://github.com/username-kamu/bluebooth.git`
2. Masuk ke folder: `cd bluebooth`
3. Install dependencies: `npm install`
4. Jalankan server lokal: `npm run dev`
5. Buka `http://localhost:3000` di browser.
